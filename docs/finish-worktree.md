# Finish Worktree Skill

## Overview
The `finish-worktree` skill is designed to streamline the completion of a task within a Git worktree. It automates the boilerplate steps required when concluding a piece of work, preparing it for review and merging.

## Features
- Exits the current active Git worktree safely.
- Prepares a draft Pull Request (PR) based on the current diff.
- Automatically detects and applies project PR templates.
- Outputs copy-pasteable Markdown summarizing the changes, making it easy to create the actual PR description on GitHub/GitLab.

## Usage
Activate this skill when you have completed a task or feature in an isolated worktree and are ready to prepare a pull request.
