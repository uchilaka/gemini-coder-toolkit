# Markdown Manager Skill

## Overview
The `markdown-manager` skill introduces an expert persona dedicated to documentation and Markdown file management.

## Features
- Ensures consistency and proper structure across all markdown files in a project.
- Enforces canonical storage formats and naming conventions.
- Assists in drafting high-quality project plans, technical documentation, and Pull Request reviews.

## Usage
Use this skill whenever you are writing or updating project plans, creating technical documentation, or generating structured PR reviews that require a polished, readable Markdown format.
